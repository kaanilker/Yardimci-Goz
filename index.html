👁️ Yardımcı Göz Nedir?

Yardımcı Göz; yaşlı bireyler veya okumakta güçlük çeken kişiler için tasarlanmış bir belge erişilebilirlik asistanıdır. Fatura, ilaç prospektüsü, resmi mektup gibi belgeleri fotoğraflayarak içeriği sade Türkçeyle öğrenebilirsiniz.

Önemli tarihleri, ödeme son günlerini ve riskleri otomatik olarak işaretler. Sesli dinleme ve sesli soru sorma özelliği sayesinde ekrana bakmadan da kullanılabilir.

Nasıl çalışır?

1. Fotoğraf Çek butonuna basın veya galeriden seçin.
2. Tesseract.js yazıları otomatik olarak okur (OCR).
3. Yapay zeka belgeyi sade Türkçeyle özetler.
4. Özeti sesli dinleyin veya sorularınızı yazın / sesli sorun.

Kullanılan teknolojiler:

Tesseract.js (OCR) Groq API Llama 3.1 8B Instant Whisper Large V3 Turbo Microsoft Edge TTS PHP HTML / CSS / JS

Tüm işlemler tarayıcınızda veya güvenli sunucumuzda gerçekleşir. Fotoğraflarınız hiçbir şekilde kaydedilmez veya paylaşılmaz.

⚠️ Metin Kalitesi Düşük

Fotoğraftaki yazılar net okunamadı. Özet yanlış veya eksik olabilir.

Daha iyi sonuç için:

• Belgeyi düz ve aydınlık bir yerde çekin
• Kamerayı yaklaştırın
• Bulanık veya gölgeli fotoğraflardan kaçının
